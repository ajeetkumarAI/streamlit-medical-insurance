# Medical Insurance Charges Prediction

A Streamlit web application that predicts medical insurance charges using a Linear Regression model.

## Project Structure

```
├── app.py                  # Streamlit app
├── train_model.py          # Train model & save pickle + JSON
├── medical_insurance.csv   # Training dataset
├── Linear_Model.pkl        # Pre-trained model (generated by train_model.py)
├── project_data.json       # Model metadata (generated by train_model.py)
├── requirements.txt        # Dependencies
├── .gitignore
└── README.md
```

## Technologies Used

- Python
- Streamlit
- Scikit-learn
- Pandas
- NumPy

## Local Setup

```bash
# Clone the repository
git clone https://github.com/ajeetkumarAI/streamlit-medical-insurance.git

cd medical-insurance-prediction

# Create virtual environment
python -m venv venv

# Activate virtual environment
venv\Scripts\activate         # Windows
# source venv/bin/activate        # Linux / macOS


# Install dependencies
pip install -r requirements.txt

# Train the model (generates Linear_Model.pkl and project_data.json)
python train_model.py

# Run the app
streamlit run app.py
```

Open http://localhost:8501 in your browser.

## Azure Deployment

1. Create a GitHub repo and push this code to `main`
2. Create Azure Web App (Python 3.12, Linux)
3. In Azure Portal → Deployment Center → Connect your GitHub repo
4. Set Startup Command in Configuration → General Settings:
   ```
   python -m streamlit run app.py --server.port 8000 --server.address 0.0.0.0
   ```
